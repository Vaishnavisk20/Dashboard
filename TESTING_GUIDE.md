# Testing Guide

Run all focused tests:

```bash
npm test
```

Run syntax/build checks:

```bash
npm run build
```

Covered:

- IC fallback.
- Stable project keys.
- Risk scoring.
- Forecast delay calculation.
- Forecast confidence.
- CSV normalization and date parsing.
- KPI count/detail consistency.
- Selected-date forecast API.
- CSV preview and duplicate-safe confirmation.
